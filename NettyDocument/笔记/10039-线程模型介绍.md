# 10039-线程模型介绍

线程模型介绍

**1**、不同的线程模式，对程序的性能有很大影响，为了搞清Netty 线程模式，我们来系统的讲解下 各个线程模式， 最后看看Netty 线程模型有什么优越性。

**2**、目前存在的线程模型有：

- 传统阻塞 I/O 服务模型
- Reactor 模式

**3**、**根据** **Reactor** **的数量和处理资源池线程的数量不同，有** **3** **种典型的实现：**

- 单 Reactor 单线程；
- 单 Reactor 多线程；
- 主从 Reactor 多线程

**4**、Netty 线程模式(Netty 主要**基于****主从** **Reactor** **多线程模型**做了一定的改进，其中主从 Reactor 多线程模型有多个 Reactor)
